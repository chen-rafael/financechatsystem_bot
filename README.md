# Telegram BOT de Sistema Bancário com Python

Um BOT desenvolvido em Python utilizando a API do Telegram com funcionalidades básicas de controle financeiro.

# Sobre

O BOT simula um sistema bancário simples, com funcionalidades como:
- Sistema de Saldo/Depósito/Saque
- Simulação de Rendimento
- Respostas de Dúvidas

Todas as interações realizadas via comandos no Telegram.

# Instalação

1. Clone o repositório ou faça o download dos arquivos.
2. Crie um BOT no Telegram utilizando o @BotFather
3. Crie um arquivo .env
4. Copie e cole o TOKEN fornecido no arquivo .env (BOT_TOKEN=SEU_TOKEN)
5. Instale as dependências (pip install pyTelegramBotAPI python-dotenv)
6. Inicie o BOT (python bot.py), procure por ele no Telegram e clique em **Start**
